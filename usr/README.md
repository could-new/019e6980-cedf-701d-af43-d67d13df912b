# PictorialFX

A responsive, single-page promotional website for a VFX design company, showcasing a dark cinematic theme and a portfolio gallery.

## Features
- **Hero Section:** Full-width cinematic banner.
- **Responsive Gallery:** A grid showcasing movie-style visual effects imagery that adapts to mobile, tablet, and desktop viewports.
- **Dark Theme:** High-contrast aesthetic typical of creative and post-production studios.

## Tech Stack
- Flutter (Web, iOS, Android, Desktop)
- Dart

## Getting Started

To run the project locally:

```bash
flutter pub get
flutter run
```
